Evaluating Macros

The "evaluate" feature allows you to view macro-definitions. Currently, evaluating macros is supported with the DWARF format only, and GNU CC 3.2.2 and above. You need to compile your programs on the highest debug info level (use the -g3 compiler option).

Example: Select Tools from the menu bar, then Evaluate. Enter __GNUC__ in the expression field and press the Evaluate button (or hit <ENTER>). The major version number of the compiler that translated the unit in scope will be displayed. To view the minor version number, evaluate the __GNUC_MINOR__ macro.

Another way of accessing this feature is to right-click on a macro in the source code view, and select the Evaluate entry from the floating menu.

Note: Parameter substitution in function-like macros is not currently supported.

Note: Symbols with the same name as a given macro, visible in the current scope of the evaluation, obscure the macro-definition.
