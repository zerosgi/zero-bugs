Introduction


This is a debugger for Linux user-mode programs written in C/C++.

Although debugging programs written in other compiled languages such as Fortran or Ada might work, this is not tested. The debugger is designed with C/C++ in mind.

The debugger handles multithreaded applications, and it supports both the older pthread library (Linux Threads) and the new NPTL (Native Posix Thread Library).

Visit the project and browse the source code at Codeplex.
