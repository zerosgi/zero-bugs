Viewing Local Variables

The debugger's window area located immediately under the menu bar and the toolbar displays the local variables (automatic variables that are visible in the current scope of execution).

To hide this area, select Locals in the View menu. To show it again, select the menu once more.

The local variables view is organized into three columns: one column for the variable name, another one for the value, and a third that displays the type.

Aggregated types such as structures and classes are shown in a hierarchical manner, in a tree

Right-clicking on a row in this window area pops up a menu with several options. You can toggle between hexadecimal and decimal representations for integers, view the variable as raw memory or set a watchpoint at the memory location that corresponds to the variable.

You can alter the value of a variable: double-click on it, then enter the new value in the edit field that pops up. Press Escape to abort, or Enter to accept the change.
